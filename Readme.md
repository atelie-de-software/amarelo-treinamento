# Projeto Exemplo
### Build
`docker-compose build projeto`
### Rodar projeto:
`docker-compose run projeto`
### Acesso
`docker-compose run projeto bash`
### Rodar testes no container
`rspec`
### Atualizar Gems no container
`bundle`
